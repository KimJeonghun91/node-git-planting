# 사용 방법

1. git 아이디 설정

    ```bash
    git config --global user.email "<깃 ID>"
    git config --global user.name "<깃 NAME>"
    ```

2. git 프로젝트 다운

    ```bash
    git clone https://github.com/KimJeonghun91/node-git-planting
    ```

3. 저장소 연결

    ```bash
    git remote add origin [your repository]
    ```

4. 실행

    ```bash
    npm start
    ```
